# raycaster-applesoft
3D Raycaster Maze program written in Applesoft BASIC

- Version: 1.0
- Language: Applesoft BASIC
- Video Mode: 40x40 Low-Resolution Graphics (GR)

DESCRIPTION
-----------
This program is a functioning "Wolfenstein 3D" style raycasting engine
written entirely in standard Applesoft BASIC. It renders a 3D 
first-person perspective of a 2D grid map. 

Features include:
- Textured environments (Dark grey ceiling, white floor)
- Multi-colored solid walls
- Pseudo-sprites (green barrels you can see past)
- Collision detection 
- An interactive exit/win condition
- Fisheye distortion correction

HOW TO RUN
----------
1. Boot up an Apple II emulator or use real Apple II hardware.
2. Ensure you are at the Applesoft BASIC prompt (the `]` symbol).
3. Paste or load the source code into memory.
4. Type RUN and press ENTER.

CONTROLS
--------

- W - Forward
- S - Backward
- A - Rotate Left
- D - Rotate Right

Note: Because the engine is turn-based and relies on Applesoft's GET 
command, the game waits for a keypress before calculating and 
drawing the next frame.

CUSTOMIZING THE MAP
-------------------
The 3D environment is generated by the DATA statements at the very 
bottom of the code (Lines 1040 - 1130). The map is a 10x10 grid. 
You can design your own mazes by changing the numbers in these lines!

Map Legend:
-  0 = Empty Space (Walkable)
-  1 = Dark Blue Wall
-  2 = Aquamarine Exit Door (Walk into this to win the game!)
-  3 = Green Barrel (Pass-through obstacle)
-  4 = Red Wall
-  5 = Purple Wall
-  6 = Medium Blue Wall

PERFORMANCE NOTES
-----------------
Because standard Applesoft BASIC is an interpreted language that 
processes complex floating-point trigonometry (SIN/COS) on a 1 MHz 
processor, rendering a single frame takes approximately 1 second.
